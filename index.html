Setting Up a Jeykyll Octopress Blog Hosted on Github

Jul 13th, 2013

The 5 Minute Setup Guideline

1. Get octopress

octopress setup

+ clone repo
+ rake install

2. Setup your github page

octopress github setup

3. Deploy

$ rake setup_github_pages

$ rake generate
$ rake deploy

4. Commit your code

$ git add .
$ git commit -m "initial deploy"
$ git push origin source

5. Configure your blog

octopress configure

6. Create a blog post

octopress configure

$ rake new_post["title"]

7. Check it out

$ rake generate

Local: $ rake preview
Github: $ rake deploy
